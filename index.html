FulDC++

A modern, fully dark direct-connect client for Windows — built on AirDC++, and happy to run right alongside it.

Download for Windows · x64 See what's new

Why FulDC++

Everything you already know from AirDC++ / StrongDC++, plus a genuinely dark UI and a stack of quality-of-life features.

🌙

True dark theme

A fully independent dark mode — title bars, menus, lists and chat — with 52 presets and a custom palette editor. Toggle live, no restart.

🤝

Runs beside AirDC++

Its own instance and profile folders, with a one-time automatic import of your existing AirDC++ settings. Try it without giving anything up.

🔌

Native plugins

Full DC++ C-ABI plugin support (DCAPI v8) — all interfaces and hooks, with a Plugins menu and settings page.

💬

Chat triggers

Keyword and regex chat automation with capture groups, away-only mode, fire stats and import/export.

🛟

Auto-recovery

A watchdog restarts FulDC++ automatically if it ever crashes or stops responding, so long sessions stay up.

🌍

Self-updating GeoIP

Built-in IP geolocation (DB-IP.com) with a configurable, self-refreshing database.
