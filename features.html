Features

FulDC++ is a full DC client — ADC and NMDC, TLS transfers, segmented downloading, autosearch and an optional Web UI. Below are the additions that set it apart from stock AirDC++.

Look & feel

Full independent dark theme

Dark title bars, menus, scrollbars, list/tree chrome, chat and status bar — independent of the OS theme. 52 colour presets, a custom palette editor, follow-OS option and per-hub tab colours. Toggles live with no restart.

GUI modernization

Inline chat find bar (F3), resizable dialogs, TaskDialog prompts, a recent-hubs dropdown, quick-connect speed-pane sparklines and an Aero connect wizard.

Status-bar enhancements

Clickable Slots/Queued panes, show/hide panels from a right-click menu, plus shared-files, favourites-online and refresh-status panels.

Vertical tabs & chat autocomplete

Optional vertical tab strip, per-hub tab colouring, and a slash-command autocomplete popup in chat.

Power features

Runs side-by-side with AirDC++

Its own instance identity and profile folders, with a one-time automatic import of an existing AirDC++ profile. Install it without disturbing your current setup.

Native DCAPI plugins

DC++ native C-ABI plugin support (DCAPI v8) — all 10 interfaces and 26 hooks, with a Plugins menu and settings page.

Chat triggers

Keyword / regex chat automation with capture groups $1–$9, away-only mode, fire statistics and import/export.

Auto-recovery & crash supervisor

A watchdog automatically restarts FulDC++ if it crashes or stops responding, including a freeze watchdog for hangs.

Group users by type

Hub user lists grouped into Ops / Bots / Users with live header counts.

Self-updating GeoIP

IP geolocation from DB-IP.com with a configurable update URL, expiry and a validated staged swap. Dedicated GeoIP settings page.

"Only secure hubs" search & pause-all

A quick-select search filter for hubs with secure transfers, and a one-click pause-all-downloads toolbar action.

Security & protocol robustness

Hardened networking

Rejection of spoofed UDP packets, side-by-side port-hijack protection, and a one-click dialog to re-pin a changed TLS keyprint.

Protocol gap fixes

ADC additions (failover, RDEX, passive PSR relay and more) and NMDC additions ($MyINFO server/fireball bits, $NickRule/$SearchRule, TTHS short search, $FailOver, $MCTo, salted-password login, $BotList).

In-client updates

Signed, self-installing updates delivered straight from this site — no manual reinstall.
